Tools
True Digivice refers to the 11 types of Digivice corresponding to each crest from Digimon Adventure.

Each type of True Digivice adds a unique visual effect to your tamer.
True Digivices can be crafted via the Nanomon (Item Craft) NPC at Dats Center. The craft materials can be dropped from various raid bosses in Tokyo-Odaiba maps.
The base version,  Digivice of Beginning Lv. 0, can be obtained from a main quest at Minato City.
You can craft an additional True Digivice using the  103-OT Digivice.
Digivice Attributes
All version of True Digivice comes with 2 options. The options work similarly to the Rings, Necklaces, Earrings, and Bracelets, meaning that they can be changed with  Option Change Stone and  Number Change Stone, can be upgraded to 200% with Digitary Power Stone, and can be renewed with Renewal Increase Stone.
The 2 Digivice options can be any of the Digimon Attributes and Digimon Elements.
Unlike the Basic Attribute option in Rings, Necklaces, Earrings, and Bracelets, the Digivice Attributes are NOT damage bonus from attribute advantage. Instead, they increase the skill damage of Digimon of said Attribute. For example, the Light Attribute will increase the skill damage of every Light Digimon that you use, regardless of the enemy type matchup.
The maximum number is 10% in Digimon Attributes and 15% in Digimon Elements.
You cannot get the same option twice.
 Digivice of Adventure Lv2 and all versions of  103-OT Digivice are the same as True Digivice, minus the tamer visual effect part.

					<div id="mw-content-text" class="mw-body-content"><div class="mw-content-ltr mw-parser-output" lang="en" dir="ltr"><p><b>True Digivice</b> refers to the 11 types of Digivice corresponding to each crest from Digimon Adventure. 
</p>
<ul><li>Each type of True Digivice adds a unique visual effect to your tamer.</li>
<li>True Digivices can be crafted via the <a href="/Nanomon_(Item_Craft)" title="Nanomon (Item Craft)">Nanomon (Item Craft)</a> NPC at <a href="/Dats_Center" class="mw-redirect" title="Dats Center">Dats Center</a>. The craft materials can be dropped from various raid bosses in <a href="/Tokyo-Odaiba" title="Tokyo-Odaiba">Tokyo-Odaiba</a> maps.</li>
<li>The base version, <span class="mw-default-size" typeof="mw:File"><a href="/File:Digivice_of_Beginning_Lv_0.PNG" class="mw-file-description"><img src="/images/6/63/Digivice_of_Beginning_Lv_0.PNG" decoding="async" loading="lazy" width="30" height="30" class="mw-file-element" /></a></span> Digivice of Beginning Lv. 0, can be obtained from a main quest at <a href="/Minato_City" title="Minato City">Minato City</a>.</li>
<li>You can craft an additional True Digivice using the <span class="mw-default-size" typeof="mw:File"><a href="/File:OT_Digivice.PNG" class="mw-file-description"><img src="/images/f/f2/OT_Digivice.PNG" decoding="async" loading="lazy" width="60" height="30" class="mw-file-element" /></a></span> <b>103-OT Digivice</b>.</li></ul>
<meta property="mw:PageProp/toc" />
<h2><span class="mw-headline" id="Digivice_Attributes">Digivice Attributes</span></h2>
<ul><li>All version of True Digivice comes with 2 options. The options work similarly to the Rings, Necklaces, Earrings, and Bracelets, meaning that they can be changed with <span class="mw-default-size" typeof="mw:File"><a href="/File:Option_Change_Stone.png" class="mw-file-description"><img src="/images/4/44/Option_Change_Stone.png" decoding="async" loading="lazy" width="32" height="32" class="mw-file-element" /></a></span> <b>Option Change Stone</b> and <span class="mw-default-size" typeof="mw:File"><a href="/File:Number_Change_Stone.png" class="mw-file-description"><img src="/images/f/f0/Number_Change_Stone.png" decoding="async" loading="lazy" width="32" height="32" class="mw-file-element" /></a></span> <b>Number Change Stone</b>, can be upgraded to 200% with <b>Digitary Power Stone</b>, and can be renewed with <b>Renewal Increase Stone</b>.</li>
<li>The 2 Digivice options can be any of the <a href="/Digimon_Attributes" title="Digimon Attributes">Digimon Attributes</a> and <a href="/Elemental_Attributes" title="Elemental Attributes">Digimon Elements</a>.</li>
<li>Unlike the Basic Attribute option in Rings, Necklaces, Earrings, and Bracelets, the Digivice Attributes are NOT damage bonus from attribute advantage. Instead, they increase the skill damage of Digimon of said Attribute. For example, the Light Attribute will increase the skill damage of every Light Digimon that you use, regardless of the enemy type matchup.</li>
<li>The maximum number is 10% in <a href="/Digimon_Attributes" title="Digimon Attributes">Digimon Attributes</a> and 15% in <a href="/Elemental_Attributes" title="Elemental Attributes">Digimon Elements</a>.</li>
<li>You cannot get the same option twice.</li>
<li><span class="mw-default-size" typeof="mw:File"><a href="/File:Digivice_of_Adventure_Lv_2.PNG" class="mw-file-description"><img src="/images/8/80/Digivice_of_Adventure_Lv_2.PNG" decoding="async" loading="lazy" width="30" height="30" class="mw-file-element" /></a></span> <b>Digivice of Adventure Lv2</b> and all versions of <span class="mw-default-size" typeof="mw:File"><a href="/File:OT_Digivice.PNG" class="mw-file-description"><img src="/images/f/f2/OT_Digivice.PNG" decoding="async" loading="lazy" width="60" height="30" class="mw-file-element" /></a></span> <b>103-OT Digivice</b> are the same as True Digivice, minus the tamer visual effect part.</li></ul>
